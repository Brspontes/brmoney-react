<h2>Usage</h2>
First clonning repository using

```bash
git clone https://github.com/Brspontes/brmoney-react.git

cd react-movie-list
```
<br />
In current folder, before compilation, restore packages with

```bash
npm install
or
yarn install
```
Run App
```bash
npm start
or
yarn start
```

Go to:  https://localhost:3000/

app opnend
